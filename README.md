# 小型校园管理平台（实训）-Campus management system

#### 介绍
小型校园管理平台（教学实训）
本项目的定位是后台集成方案，不太适合当基础模板来进行二次开发。因为本项目集成了很多你可能用不到的功能，会造成不少的代码冗余。如果你的项目不关注这方面的问题，也可以直接基于它进行二次开发。

#### 软件架构
使用node.js


#### 安装教程

1.  npm install    #下载插件数据包
2.  npm run dev    #运行

#### 使用说明

1.  请自行更改school-sever\db\config.js文件中的数据库链接。
